### TERMUX
```sh
$ pkg upgrade -y && pkg update -y
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/MaddeXD/WhatsApp-Bot-Bug
$ cd WhatsApp-Bot-Bug
$ pkg install yarn
$ yarn
$ npm i 
$ npm start
```
